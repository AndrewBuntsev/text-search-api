# text-search-api

## Available Scripts

In the project directory, you can run:

### `npm start`
Runs the 'node index.js' command.<br>

### `npm run dev`
Runs the 'nodemon index.js' command.<br>

### `node testEndpoints.js`
Runs the test endpoints on the 9998 port.<br>

## Endpoints

### http://localhost:9999/
### http://localhost:9999/searchText
